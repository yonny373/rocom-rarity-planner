## 基础模型
### 官方公告综述
#### 1.星光值与赛季连锁进度
赛季奇遇事件是自开服以来存在的机制，是洛手的核心玩法之一。
在早期，部分洛学家提出污染值累积模型、污染值-星光值线性关联假设，可简单概括为：通过成功捕捉精灵累计污染值≈星光值到达阈值触发S1赛季奇遇事件“噩梦枷锁”（经多方查证，其起源暂未明确）。该模型虽存在偏差，但实质今日仍在被小部分洛学家沿用。
直至S1中旬官方在公告<sup>[&#91;2&#93;](#refer-anchor-2)</sup>中明确了成功捕捉精灵时提供赛季连锁进度，否定了赛季连锁进度与星光值挂钩，肯定了赛季连锁进度与精灵捕捉难度正相关。奇遇的赛季连锁进度按精灵捕捉难度累计的模型完成初步建立。
